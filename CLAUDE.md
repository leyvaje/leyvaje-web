# design

Start to be a whole filled image from me if there is not one take a random one de mi Instagram y enviarlo en el background @leyvajex

El Background debe de ser monocromo, debe tener una animación aleatoria matemática que que tome en cuenta la aceleración del del Mouse encima de la pantalla por ejemplo utiliza conceptos como el golden ratio o euclides algoritmo para animar un emoji. Agrega un botón para guardar el arte, ya que con el movimiento debe de quedarse guardado hasta que el usuario abandone la página.

# musica

Cómo definición de la parte visible siempre el Screen View no importa si el usuario está haciendo un scroll Down o o está viendo un artículo o algo siempre en la parte visible puede el usuario hacer Beats con samples como Drums High hats. Cada vez que el usuario presiona una tecla también en la pantalla se anima. Algo sale una forma geométrica aleatoria como referencia. Utiliza la website patatap.

Por otro lado, también incluirá un teclado virtual que también con las teclas. Se puede tocar una escala completa incluyéndose semi tonos por ejemplo la tecla de es DO, la tecla R es dos sostenido.

Para que no haya conflicto de teclas, podemos utilizar la hilera o los números para los Beats del uno al cero.
